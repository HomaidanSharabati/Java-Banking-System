# Java Banking System 🏦

A console-based banking application developed in Java, demonstrating strong Object-Oriented Programming (OOP) principles.

## Features
* Secure file-based data persistence (saving user data locally).
* Custom admin dashboard for system management.
* Multi-account financial operations: Deposits, Withdrawals, and Transfers.

## Tech Stack
* **Language:** Java
* **Concepts:** OOP, File Handling, Data Structures.
